Job Market Analytics Dashboard is an interactive data analytics application built using PostgreSQL, Python, Streamlit, and Plotly.

The dashboard analyzes AI & Data Science job market data to provide insights into:

Salary trends
Job demand
Skills demand
Hiring patterns
Remote work opportunities
Industry hiring trends

The project contains 10,345 job records and allows users to explore the job market using dynamic filters.
